This slightly ugly sample app demonstrates how to set up windows and populate them with goui. What it doesn't demonstrate is how to make your apps look pretty.

Every goui window is just a web view that you can populate with any HTML, CSS and JavaScript you like, so making it look pretty is up to you. This example uses jQuery to handle some of the JS goop automagically, but you can use any JavaScript libraries you like.

Doing things in the background

Most GUI systems will only work when called from the main thread, and hog the main thread for the life of your program. That's still sort-of true with goui, but it's super easy to spawn as many goroutines as you like to keep things ticking along in the background.
